# 🏨 Hotel Gunja – Hotel Booking & Management System

## 📌 About the Project
Hotel Gunja is a full-stack, responsive web-based plus  hotel management system developed using Laravel. It automates hotel operations such as room booking, billing, restaurant services, event management, and customer handling.

---

## ⚙️ Key Features
- Room booking and management system  
- User authentication and role-based access  
- Admin dashboard with full control  
- Booking approval system (Receptionist)  
- Billing and payment management (Cashier)  
- Restaurant and event management  
- Gallery and homepage content management  
- Review and feedback system  

---

## 👥 User Roles & Access Control

### 🟢 Admin
Full control of the system. Admin can manage:
- Rooms and room categories  
- Homepage banners and images  
- Gallery content  
- Events and event halls  
- Restaurant and dining management  
- Reviews and feedback  
- Billing and invoices  
- Image uploads for rooms, gallery, and restaurant sections  

---

### 🟡 Receptionist
Responsible for managing customer bookings:
- Create and manage reservations  
- Approve or reject booking requests  
- Handle check-in and check-out processes  

---

### 🔵 Cashier
Responsible for financial operations:
- Generate and manage bills  
- Process payments  
- Maintain transaction records  

---

## 🛠️ Tech Stack
- Laravel (PHP Framework)  
- MySQL Database  
- HTML, CSS, JavaScript  
- Bootstrap  

---

## 🎯 Purpose
To digitalize hotel operations and provide a centralized system for efficient management of hotel services.
